# Сведения об образовательной организации - ООО "АСО АВТОШКОЛ"

Статический сайт для GitHub Pages по ТЗ `ТЗ_на_сайт_сведения_об_организации_АСО_АВТОШКОЛ.docx`.

## Что внутри

- `index.html` - все обязательные разделы из ТЗ на одной доступной HTML-странице.
- `styles.css` - визуальный стиль в духе текущего сайта АСО: светлая premium-основа, красно-синие акценты, стеклянная шапка.
- `script.js` - версия для слабовидящих, поиск по сведениям, мобильное меню и печать.
- `docs/` - папка для PDF-сканов документов.
- `.github/workflows/pages.yml` - деплой статического сайта на GitHub Pages.

## Документы

PDF-файлы не приложены к ТЗ, поэтому на сайте они отмечены как "Ожидается PDF".
Когда сканы будут готовы, положите их в `docs/` и замените статус на ссылки в `index.html`.

## Локальный просмотр

Откройте `index.html` в браузере. Сервер и сборка не требуются.

## GitHub Pages

Репозиторий можно публиковать через GitHub Pages из ветки `main` или через workflow `pages.yml`.
